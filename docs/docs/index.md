# Trading Crew

**Multi-agent crypto trading system powered by CrewAI.**

Trading Crew uses specialized AI agents to handle different aspects of
cryptocurrency trading — from fetching market data and computing indicators,
to generating signals, managing risk, and executing orders.

## Key Features

- **Multi-Agent Architecture** — Each agent has one job, crews coordinate them
- **Multi-Exchange** — 100+ exchanges via CCXT
- **Risk-First** — Position sizing, stop-loss, portfolio limits, circuit breakers
- **Paper Trading Default** — Safe to experiment with
- **Pluggable Strategies** — Easy to add your own
